Welcome

This site contains code to run the DANSE algorithm (3 different implementations) in the MATLAB environment. In order to initially run the code, just download the the files via the links on the left. Once downloaded and unpacked, simply run the following commands : 

>> addpath(genpath(pwd))
>> batch_run

and you're off.

Main Algorithms

The current code contains 3 different implementations of the DANSE algorithm. The folder contents are described below :

  • DANSE
    • DANSE.m - Generates broadcast signal, calls filter update sub function, and calculates cost
    • DANSE_filt_update.m - Updates node-specific filter
  • TDANSE
    • TDANSE.m - Generates broadcast signal, calls fusion and diffusion flow sub-functions, calls filter update sub-function, and calculates cost
    • TDANSE_rooted_ff.m - Fusion flow (data aggregation)
    • TDANSE_rooted_df.m - Diffusion flow (data dissemination)
    • TDANSE_filt_update.m - Updates node-specific filter
  • TIDANSE (fully connected)
    • TIDANSE_fc.m - Generates broadcast signal, calls filter update sub-function, and calculates cost
    • TDANSE_filt_update.m - Updates node-specific filter
  • TIDANSE (in a tree topology)
    • TIDANSE_tree.m - Generates broadcast signal, calls fusion flow sub-functions, calls filter update sub-function, and calculates cost
    • TIDANSE_rooted_ff.m - Fusion flow (data aggregation)
    • TDANSE_filt_update.m - Updates node-specific filter
